
# Ration card management system
 “RATION CARD MANAGEMENT SYSTEM” is an efficient system to ease the customers to buy groceries from
distributors allotted by the government.
 It eradicates the corruption and maintains the transparency between government local bodies and distributors.
Frontend is designed with HTML,CSS, JavaScript, Streamlit.
 Backend is designed with MySql and Python

## Deployment

To run this project on vs code

```bash
pip install streamlit
pip install streamlit-authenticator==0.1.5
pip install MySQL
pip install mysql-connector-python
pip install plotly
py -m streamlit run app.py 
```


## Features

- Admin interface
- shopkeeper interface
- selling products 
- Pie charts
